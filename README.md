# Community Library

Aplicação para catalogação e indexação de obras com base nas Diretrizes para Indexação de Obras Estético‑Literárias (DIEL).

## Objetivo (MVP)

Entregar rapidamente uma aplicação web para que um bibliotecário consiga:

- Cadastrar uma obra
- Consultar/listar obras
- Editar e excluir
- Registrar indexação (DIEL) progressivamente

A indexação DIEL será incorporada em fases, começando por um conjunto mínimo de campos e evoluindo conforme feedback do usuário.

## Stack / Tecnologias

### Backend
- Java 21
- Spring Boot
- Spring Data MongoDB
- MongoDB

### Frontend
- Node.js 24.11.0
- Vite
- React
- RSuite (UI)
